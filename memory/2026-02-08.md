## Complete Agent API Reference

### Menu Navigation (PrimalKardsMenu)
```javascript
PrimalKardsMenu.getState()           // Current screen, login status, NFTs
PrimalKardsMenu.skipLogin()          // Skip to basic avatar selection
PrimalKardsMenu.loginWithToken(token) // Login with killabears.com token
PrimalKardsMenu.getNFTs()            // Get NFT list after login
PrimalKardsMenu.selectNFT(index)     // Select NFT avatar
PrimalKardsMenu.confirmNFTSelection() // Confirm NFT and start
PrimalKardsMenu.selectAvatar(index)  // Select basic avatar (0-4)
PrimalKardsMenu.startGame()          // Start game with selection
```

### Gameplay (PrimalKards)
```javascript
PrimalKards.getState()               // Full game state (HP, AP, hand, board, etc.)
PrimalKards.getActions()             // Available moves
PrimalKards.playCard(handIndex)      // Play card from hand
PrimalKards.attack(attackerIdx, targetIdx) // Attack (-1 for face)
PrimalKards.endTurn()                // End current turn
PrimalKards.useInstant(handIdx, targetIdx) // Cast instant spell
PrimalKards.equipCard(handIdx, boardIdx)   // Equip to creature
PrimalKards.allOutAttack()           // Avatar special ability
```
