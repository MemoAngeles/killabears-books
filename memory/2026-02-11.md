## Lore Work: Status Effects Overhaul

Memo clarified:
- Book 1 ends at the cliffhanger (Mops meets Ursoth) — no ch 10-11 needed
- Book 2 will have FomoBit rescue with MotherBear ship + game lore integration

**ROOT Mechanic Redesign:**
Created `/Lore/game_mechanics_status_effects.md` with new system:

1. **ROOT = Physical Restraint** (not biological!)
   - Vines, ropes, webs, ice, chains, nets, tar, gravity
   - Countered by: Fire (burns bonds), Strength (break free), Cleanse

2. **POISON = Biological Damage**
   - Venom, toxic plants, chemicals, disease
   - Countered by: Antidote, Heal, Cleanse

3. **NEW: FUNGAL/SPORE = Transformation**
   - Progressive: weakens → transforms into Fungal Husk
   - Countered by: Fire, Cold, Cleanse (fungi hate heat!)

4. **Makarasas now use both:**
   - ROOT (webs = physical)
   - POISON (venom = biological)
   - Makes them more dangerous and logical!

Key insight: Counters now make intuitive sense (fire melts ice, burns webs/vines)
